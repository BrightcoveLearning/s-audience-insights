Integration setup

Follow the steps below to set up the integration to your S3 bucket.

  1. Login to Studio.
  2. Use the left navigation to go to the Integrations module:
  3. Click Customer Data Platforms to open it:
  4. Click Add Integration
  5. Select Custom S3 from the Destination dropdown:
  6. In the Add Integration dialog, enter your AWS access key and secret access key, S3 path, choose daily or weekly report exports, and enter one or more email addresses to be notified in case of export errors:
    If you are not sur how to generate/find the AWS information, consult the AWS Documentation.
  7. Click Save to create the integration.

Note that you can edit the integration details later by returning the Customer Data Services home page and clicking the .
