Requirements

  • Being an user with the admin role in Audience Insights.
  • Have a HubSpot account.
  • Create a private app in your HubSpot account.
  • Have a HubSpot user with Super admin permissions to create new private apps.

HubSpot Private App Setup

To set up your HubSpot Private App, follow these steps:

  1. Log in to your HubSpot account.
  2. Click settings icon in the main navigation bar.
  3. In the left sidebar menu, navigate to Integrations > Private Apps and click Create a private app.
  4. On the Basic Info tab, configure the details of the app. The name is required and can be anything you like as long as it is unique within your HubSpot account. The logo and description are optional.
  5. Click the Scopes tab and select the Read and Write options for the crm.objects.contacts and crm.schemas.contacts items under the CRM section. Also select the Request option for the oauth item under the Standard section.
  6. Click Create app in the upper right hand corner of the screen.
  7. In the dialog box, review the info about your app's access token, then click Continue creating.
  8. Once you have created the app, you will see a dialog which contains the unique token for the private app. Click Copy. You will need to enter this value when adding the HubSpot Publish Integration.

Add Publish Integration

To publish a HubSpot integration from audience insights, follow these steps:

  1. Login to Video Cloud Studio.
  2. Open the Integrations module.
  3. Click Customer Data Platforms.
  4. Click Add Integrations or click Connect customer data platform.
  5. Select HubSpot from the Destination dropdown.
  6. Fill out the form with the Access Token from the HubSpot Private App.
  7. You can choose either Customer Id or Email field to connect accounts from Audience Insights with HubSpot.
  8. Notice that the automated export option controls how often it is published to the HubSpot integration.
  9. Click Save.

Supported Properties
