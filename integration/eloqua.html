Overview

With this integration, the Contacts object in Eloqua will be updated.

Publish integrations can only update users that exist in both systems. It will not create new contacts in Eloqua. You can align users using their email or the Eloqua Contact id.

Requirements

  • Have viewer tracking enabled on your Video Cloud account.
  • Being a user with an admin role in Audience Insights.
  • Have an Eloqua account.
  • Have an Eloqua user with permissions to install aplications.
  • Install one of the two possible applications in your Eloqua instance.

Installing the application on your Eloqua Instance

To Install one of the two possible applications Click on the appropriate link and follow the onscreen directions to install the Brightcove Publish Integration for Eloqua app to your Eloqua account.

Brightcove Publish Integration for Eloqua (Audience Insights)

Brightcove Publish Integration for Eloqua (Videocloud)

Both applications are identical with the exception of where the user will be redirected after installing the application and granting authorization.

Add integration

  1. Log in to Video Cloud Studio.
  2. Open the Integrations module.
  3. Click Customer data Platforms

  4. Click Add Integration
  5. Select Eloqua from the Destination dropdown.
  6. Fill out the Form and click Save
