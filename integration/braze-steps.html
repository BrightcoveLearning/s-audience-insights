Set up Braze

To set up Braze, follow the steps below:

  1. Log into the Braze dashboard.
  2. On the menu on the right hand side, click Developer Console.
  3. Click New API key in the right hand corner.
  4. Give the key a name and check the box for user tracking.
  5. Save the key. It should create a key following the format: xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx

    This is the key required in the Essential Insights Settings page.
