Overview

This document details the properties generated by the Audience Insights reports feature. These properties can be published to partner integrations such as Salesforce, Hubspot, Eloqua, or Sailthru.

This guide will aid in the effective use of data export for enhanced audience insights and targeted marketing strategies.

Property Setup

Each publishing integration has its own way of setting up publish properties.

S3

Wicket hosts a S3 bucket where reports end up. When they add an S3 integration in the UI, we set up our policy to grant them limited access to that S3 bucket. There is also a policy they need to add to their AWS users so that they can access our bucket.

There is no need to set up properties for an S3 export, since the CSV file is copied over as-is.

Salesforce, Hubspot, Eloqua, and Sailthru

Properties are stored in the dynamo table ENV-customer-publish-integrations. The publish integrations UI (from scorecard or videocloud) has an Integration Properties tab on which the properties can be selected and saved automatically in the dynamo table.
