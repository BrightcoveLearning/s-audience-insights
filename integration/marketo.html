Overview

Audience Insights generates daily data at the user level for subscription-based accounts and accounts that do not require a subscription.

These values can be transferred to your Marketo application using this integration.

Publish integrations can only update users that exist in both systems. It will not create new contacts in Eloqua. You can align users using their email or the Eloqua Contact ID.

Requirements

  • Have viewer tracking enabled on your Video Cloud account.
  • Being a user with an admin role in Audience Insights.
  • Being a Marketo user with admin privileges

Create a Marketo Web Service

A web service must be created in the Marketo to obtain API credentials so that you can make authenticated calls to the Marketo instance.

  1. In Marketo, navigate to the Admin menu, and Click Users and Roles.
  2. Select the Roles tab and Click on New Role.
    Assign the following API permissions:

    • Read-Only Assets
    • Read-Write Custom Object
    • Read-Write Custom Object Type
    • Read-Write Lead
  3. Select the Users tab and Click Invite New User.
    Enter the required fields and Click Next.
  4. Select the API Only option, Choose the Role you created in the previous step and click Next.
  5. Click Send
    Since you are creating an API only user, no email invitation is actually sent.
  6. Go back to the Admin menu and Click Launchpoint.
  7. Click on the New menu and select New Service.
    Enter the required fields, choosing Custom for the Service and the API Only User created in the previous step.
    Click Create.
  8. Click View Details for your new service to view the Client ID and Client Secret.
    Make sure to save these credentials since you will need to enter both values when adding the Marketo publish integration.
  9. Back on the Admin menu, click on Web Services.
  10. Find the Endpoint in the REST API box.
    Make sure to save the Endpoint and Identity URLs. You will also need these values when adding the Marketo publish integration.

Audience Insights Publish Integration Setup

Once the web service has been created in the Marketo instance, a publish integration can be added to the Audience Insights system.

Requirements

  • User with the 'admin' role in the Audience Insights scorecard
  • Marketo account
  • Marketo Web Service installed in Marketo instance

Add Publish Integration

  1. Log in to Video Cloud Studio.

  2. Open the Integrations module.

  3. Click Customer Data Platforms

  4. Click Add Integration

  5. Select Marketo from the Destination dropdown.
    Fill out the Form and Click Save

    Please note:

    • The Client ID, Client Secret, API Endpoint URL and API Identity URL are all obtained from Marketo when creating the Web Service.
    • The Customer identifier can be selected as either Customer Id or Email (for Audience Insights) or just Customer Id for Videocloud.
    • The Automated Export option controls how often there should be an automated publish to the Marketo integration.

Viewing lead custom object fields in Marketo

Marketo custom objects are used to track additional attributes for leads. This publish integration creates a custom object in Marketo named Brightcove that includes fields for all of the integration properties which are selected to be published.

The specific attributes can be selected on the configuration modal for the Marketo Publish Integration in the Audience Insights settings page. Click the Integration Properties tab and choose which attributes are active and should be synchronized to Marketo.

The custom object fields are automatically created by the Marketo publish integration and do not need to be manually created in Marketo.
Please note - if a field is created (because it is in the Active column above) and later unselected (and moved to the Inactive column), the custom object field is not deleted from Marketo.

To view these fields follow the next process:

  1. Marketo from the Admin menu by navigating to Database Management > Marketo Custom Objects and clicking on the Brightcove Integration custom object.
  2. Fields tab Click on the Fields tab to view all fields that have been defined for the Brightcove Integration custom object.
