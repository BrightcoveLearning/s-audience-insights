Overview

The Test Integration feature for Contact Integrations allows you to verify the connection between Brightcove and your Marketing Automation Platform (MAP).

This feature provides feedback on errors to help identify issues and ensures successful test data visibility in the MAP.

Requirements

  • You must have video view data in the Brightcove system.
  • Video views must be attributed to a Viewer.
  • For certain integrations, you must have existing Contacts in your MAP instances.

    • Each MAP has a different process for adding these objects; please refer to your integration's documentation for details.

Customer ID and Viewer Token

Customer ID is the identifier for a Contact in the MAP instance. These will be detailed below for each integration.

Certain integrations offer a way to create Contact objects in the target MAP through Lead Forms. Players configured with these Lead Forms will attribute video views to a Viewer Token, instead of a Customer ID. These Viewer Tokens will be translated to a Customer ID before the video view data is sent to a MAP so that the data is attributed to the Contact created by the Lead Form.

For more details on how to add a Lead Form to the Player, see Adding a Lead Form.

Identifying your ID

Braze

  1. Log into Braze.
  2. In the left side bar, hover over Audience.
  3. Click Search Users.
  4. Search for a user by email, alias, external ID, etc.
  5. The Contact ID will be the User ID be on the top card.

Eloqua

Find Contact ID

  1. Log into Eloqua.
  2. Hover over the ID card icon in the top navigation bar.
  3. Click Contacts.
  4. In the search bar, search for a contact, e.g, by email.
  5. Double-click on the contact in the table.
  6. Click Field Details.
  7. In the dropdown box, select All Contact Fields.
  8. The Contact ID will be the Eloqua Contact ID.

Hubspot

Find Contact ID

  1. Log into Hubspot.
  2. In the right navigation bar, hover over CRM.
  3. Click Contacts.
  4. In the search bar, search for a contact, e.g, by email.
  5. The Contact ID will be at the end of the URL.

Marketo

Find Contact ID

  1. Log into Marketo.
  2. In the top navigation bar, click Database.
  3. In the search bar, search for a contact, e.g, by email.
  4. In the search bar, search for a contact, e.g, by email.
  5. The Contact ID will be in the ID column of the Leads table.
    6. Also, note the Created date.

S3

Find Contact ID

  1. Log into Brightcove Video Cloud.
  2. In the left navigation bar, click the Analytics module.
  3. In the left navigation bar, click Performance.
  4. Group by Viewer.
  5. Pick a Contact ID from the Viewer column.

Salesforce

Find Contact ID

  1. Log into Salesforce.
  2. In the top navigation bar, click Contacts.
  3. Click a contact in the table.
  4. The Contact ID will be in the URL after "/Contact/"

Find Viewer Token

  1. Create a Brightcove Player with Viewer Tracking and Lead Form for your desired integration enabled.
  2. Play the published video using this specific Player.
  3. Create a contact with the Lead Form.
  4. Right-click on the Player.
  5. Click Player Information.
  6. The Viewer Token will be shown as "Viewer ID".

Setup Test

  1. Log in to Video Cloud.
  2. Navigate to the Integrations module and Select Customer Data Platforms.
  3. Click the Test Integration icon for your desired integration.
  4. Set Contact to a Contact ID and click Start test.
    If you are using Marketo, Set the Start Date to a few days before the Created date of the target Contact.

Troubleshooting

If you encounter any issues using this feature, please take into account these next steps:

Unable to Authenticate with Vendor

  • Confirm that the credentials provided are correct.
  • You can test your credentials by manually calling the MAP-specific API.

Contact Does Not Exist in MAP

  • Confirm that the target Contact exists in the MAP instance and that the correct ID is being provided.
  • Refer to the Supported Integrations section above to see how to find the ID in the MAP instance.
  • If the integration uses Viewer Token, confirm that the Viewer Token is linked to a Contact.

    • Eloqua
    • Marketo
    • Hubspot

    If you are using Marketo, please make sure that 'filterType' is equal to 'cookies'.

    filterType = 'cookies'

Contact Does Not Exist in Audience Insights

  • Confirm that video view data exists in the Brightcove system and is attributed to the target Contact.
  • Refer to Collecting Viewer ID to see how to set up Viewer Tracking in the Player.

Failed to Sync/Publish Contact

If the above steps do not fix the issue, please contact Brightcove support.
